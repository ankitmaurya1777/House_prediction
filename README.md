# 🏠 California House Price Prediction

A Machine Learning project that predicts California housing prices based on various demographic and geographical features. This project demonstrates data preprocessing, exploratory data analysis (EDA), feature engineering, model building, and evaluation using regression techniques.

## 📌 Project Overview

The goal of this project is to build a predictive model that estimates house prices in California using housing-related features. The project follows a complete Machine Learning workflow, from data cleaning to model evaluation.

## 🎯 Objectives

- Analyze housing data and identify key factors affecting house prices.
- Perform data preprocessing and feature engineering.
- Train and evaluate regression models.
- Generate accurate house price predictions.
- Apply Machine Learning techniques to solve a real-world problem.

### Data Preprocessing
- Handling missing values
- Feature selection
- Feature scaling
- Data transformation

### Model Development
- Train-test split
- Model training using regression algorithms
- Hyperparameter tuning
- Performance evaluation

## 📈 Evaluation Metrics

The model performance was evaluated using:

- R² Score
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)

## 🔍 Key Insights

- Median income showed a strong relationship with house prices.
- Geographic location significantly influenced housing values.
- Proper feature scaling improved model performance.
- Data preprocessing played a critical role in prediction accuracy.
